# Проект: mesto-react

------

Небольшой лэндинг для Тренировки React

### Функционал:
* добавление/удаление карточки
* лайк карточки
* редактирование пользователя

### Новые фичи использованые в работе:
* подписка на контекст
* react-router
* хуки
* классовые компоненты

### Нужно добавить:
* валидацию форм
* интерактивные компоненты загрузки
